Token do GitHub (necessário para publicar)

O token é salvo apenas neste navegador, nunca no servidor.

Como gerar o Token do GitHub:
  1. Acesse github.com e faça login na conta seduc-afranio
  2. Vá em Settings → Developer Settings → Personal Access Tokens → Tokens (classic)
  3. Clique em "Generate new token (classic)"
  4. Dê um nome (ex: SME-Admin), selecione o escopo repo e clique em Generate token
  5. Copie o token gerado e cole no campo acima
